# 🎮 Tic-Tac-Toe AI

## 📌 Project Overview

This project implements an AI-powered Tic-Tac-Toe game where a human player competes against an intelligent AI opponent.

The AI uses the **Minimax Algorithm**, a classical Artificial Intelligence search algorithm that evaluates all possible game states and chooses the optimal move. This makes the AI extremely difficult to defeat.

This project was developed as part of the **Artificial Intelligence Internship Program at MyDailyWork**.

## 🌟 Features

* Human vs AI gameplay
